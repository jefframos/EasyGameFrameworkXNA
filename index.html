EasyGame Framework

O EasyGame Framework foi criado a partir de um experimento em XNA. Foi criado um pequeno case utilizando uma estrutura semelhante ao framework para ActionScript 3.0 de mesmo nome, o resultado foi tão satisfatório que acabou virando um framework. Inicialmente desenvolvido em ActionScript 3.0, ele foi baseado em duas engines para jogos em Flash, a Flash Punk e a Flixel. Após muito desenvolver nelas, acabei construindo meu próprio Framework utilizando estruturas semelhantes as utilizadas em ambas, tais como:

  • Sistema de Camadas
  • Sistema de Janelas
  • Sistema de Partículas
  • Entidades
  • Sistema simplificado de colisões
  • Suporte para elementos de interface ***

Rufus and the Magic Mushrooms

Rufus and the Magic Mushrooms foi um jogo que surgiu de um experimento em XNA. O jogo é baseado em um case feito na Global Game Jam 2012. Após experimentos iniciais com a parte do update e render do XNA, foi iniciado um pequeno game com o personagem totalmente reformulado. Conforme as features eram adicionadas, o jogo começou a tomar forma e acabou evoluindo para um jogo de verdade e posteriormente foi submetido para a Windows Phone Market.
